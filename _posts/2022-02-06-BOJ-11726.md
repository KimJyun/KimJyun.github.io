**[내 코드]**


>DP 문제<br/><br/>
>(주의) <br/>d[i-2] 의 경우 (1x2) // (2x1) 의 두 경우가 있다고 생각했으나 <br/>d[i-1]에서 (1x2) 의 경우를 이미 고려했으므로 한 경우로 쳐야한다.


```python
n = int(input())

d = [0] * 1001

d[1] = 1
d[2] = 2

for i in range(3, n+1):
    d[i] = ( d[i-1] + d[i-2] ) % 10007

print(d[n])
```
